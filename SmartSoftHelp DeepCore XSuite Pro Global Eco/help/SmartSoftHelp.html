NetCore代码生成{Ado.Coding} 菜单

*NetCore 原生SQL经典三层架构代码快速生成>>>>>

NetCore平台如下操作:

1.新建解决方案

2.新建 DBUtility(类库->这里放数据库操作类DbHelper)

3.新建 Common(类库->公共类库-类似smart_common_plus)

4.新建 Model(实体层-类库)

5.新建 DAL(数据接口层-类库->添加对Common，DBUtility，Model的引用)

6.新建 BLL (业务逻辑层-类库->添加对DAL的引用)

7.新建 Web项(UI-Web展示层->同时添加对DBUtility，Common，Model，DAL，BLL的引用，生成的UI文件添加到UI目录)

生成的代码分别对应添加(复制到对应目录下)到项目中即可

NetCore平台如下操作：

1.目录SmartOpenCode\\SmartCode 请使用 Microsoft Visual Studio 2015或以上版本 双击打开 SmartCode.csproj

2.启动程序SmartSoftHelp DeepCore XSuite.exe

3.返回 Visual Studio 调试->附加到进程SmartSoftHelp.exe

4.返回SmartSoftHelp DeepCore XSuite.exe > 工具 > OS 开源SmartSoftHelp DeepCore XSuite开源生成{OpenSource} 菜单

5.运行程序SmartCode源码，添加 SmartCode 程序断点即可调试程序了

6.开源代码重新编写以后，发布编译生成的SmartCode.dll，替换运行目录的原来文件即可
